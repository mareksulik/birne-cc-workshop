Čo je Claude Code

Nie je to chatbot v prehliadači.

Je to AI agent vo vašom počítači. Číta vaše súbory, píše kód, spúšťa príkazy.

Kľúčový rozdiel oproti ChatGPT alebo Claude.ai v prehliadači. CC beží priamo na vašom počítači, má prístup k súborom, vie spúšťať príkazy. Nie je to len Q&A - je to agent ktorý koná.

Terminál a inštalácia

60 minút
Začíname prvý blok. Cieľ: každý bude mať nainštalovaný a funkčný Claude Code. Začneme od základov terminálu.

Čo je terminál

Terminál = textové rozhranie pre váš počítač.

Píšete príkazy, dostávate odpovede. Chatovanie s počítačom.

Analógia: terminál je ako SMS s počítačom. Namiesto klikania na ikony píšete textové správy a počítač odpovedá textom. GUI = grafické rozhranie (to čo poznáte), CLI = príkazový riadok.

Prečo terminál

  • Automatizácia - opakujúce sa úlohy jedným príkazom
  • Presnosť - príkazy sú jednoznačné
  • Sila - veci čo v GUI nejdú
  • AI agenti bežia v termináli - to je to hlavné
Motivácia prečo sa oplatí prekonať počiatočnú bariéru. Andrej Karpathy (ex-Tesla AI, OpenAI) hovorí: CLI sú super pre AI agentov, lebo agenti vedia natívne používať terminálové nástroje. Posledný bod je kľúčový - ak chcete používať AI agentov, terminál je nutnosť.

Ghostty - moderný terminál

  • Rýchly, pekný, jednoduché nastavenia
  • Natívny na macOS aj Linux
  • GPU-akcelerovaný rendering
  • Témy, fonty, splitovanie okien

ghostty.org

Prečo Ghostty a nie default Terminal.app alebo cmd.exe. Ghostty je moderný, rýchly, pekný. Účastníci by ho mali mať predinstalovaný. Alternatívy: iTerm2 (Mac), Windows Terminal (Windows).

Základné príkazy

pwd

kde som? (print working directory)

ls

čo tu je? (list)

cd workshop

presun sa (change directory)

mkdir projekty

vytvor priečinok (make directory)

Nechať účastníkov skúšať každý príkaz. Po každom fragmente zastaviť a overiť že všetci rozumejú. pwd je najdôležitejší - vždy vedieť kde ste.

Cvičenie: Navigácia v termináli

  • Otvorte Ghostty
  • Napíšte pwd - kde ste?
  • Napíšte ls - čo vidíte?
  • Vytvorte priečinok: mkdir workshop
  • Presuňte sa doň: cd workshop
Prvé hands-on cvičenie. Chodiť medzi účastníkmi, pomáhať. Typické problémy: medzery v názvoch priečinkov, veľké/malé písmená. Dať dostatočný čas - toto je základ na ktorom staviame.

CLAUDE.md

Pamäť projektu - inštrukcie pre Claude Code.

Ako projektový brief pre AI agenta. CC si ho prečíta pri každom spustení.

  • Definovať tón komunikácie a cieľovú skupinu
  • Uviesť štruktúru projektu a dôležité súbory
  • Nastaviť formát výstupu (jazyk, dĺžka, štýl)
CLAUDE.md je ako projektový brief pre AI agenta. Každý projekt by mal mať svoj. CC ho automaticky načíta pri spustení - nemusíte ho zakaždým opisovať.

Memory - dlhodobá pamäť

  • Pamäť naprieč konverzáciami - CC si pamätá vaše preferencie
  • Automatická - CC sa učí z vašich interakcií
  • Editovateľná - viete ju upraviť alebo vymazať
  • "Pamätaj si že vždy používam slovenčinu"
Memory je osobná, CLAUDE.md je projektová. Obe sú dôležité. Príklad: ak poviete CC "pamätaj si že som marketér v e-commerce", bude to vedieť aj v budúcich konverzáciách.

Cvičenie: Váš prvý projekt

  • Vytvorte priečinok pre váš projekt
  • Požiadajte CC: "Vytvor CLAUDE.md pre môj nový projekt"
  • Definujte cieľ, cieľovku, tón komunikácie
  • Otestujte: "Napíš email podľa CLAUDE.md"
Účastníci si definujú vlastný marketingový projekt. Kto nevie, navrhnúť: email kampane pre e-shop, social media pre kaviareň, newsletter pre SaaS. CC by mal generovať obsah konzistentný s CLAUDE.md.

Slash commands

/help

zobraziť nápovedu

/clear

vyčistiť konverzáciu

/compact

zmenšiť kontext (uvoľniť pamäť)

/model

zmeniť AI model

Quick reference. /compact je asi najdôležitejší - keď CC začne "zabúdať", compact zhustí kontext a uvoľní miesto. /clear vymaže celú konverzáciu.

Kontextové okno

  • Kontext = pamäť v rámci jednej konverzácie
  • Má obmedzenú veľkosť - ako RAM v počítači
  • Čím viac toho CC spracuje, tým menej miesta zostáva
  • /compact = zmenšiť a uvoľniť miesto
Analógia: kontext je RAM vášho AI asistenta. Keď sa zaplní, začne zabúdať staršie veci. /compact zhustí konverzáciu a uvoľní miesto. Preto je dobré robiť úlohy postupne a nie všetko naraz.

Tretí blok. MCP = Model Context Protocol. Spôsob ako CC prepojiť s externými nástrojmi. Toto je pokročilá téma ale ukážeme ju jednoducho.

Čo je MCP

Model Context Protocol

Spôsob ako CC prepojiť s externými nástrojmi a službami.

Analógia: MCP je ako USB port pre AI.

MCP je USB port pre AI. Tak ako USB porty umožňujú pripojiť myš, klávesnicu, tlačiareň - MCP umožňuje pripojiť Drive, Slack, Notion, atď.

60 minút
Posledný blok. Od verziovania po bežiacu aplikáciu. Cieľ: každý bude mať projekt na internete.

Prečo verziovať

  • História zmien - vrátiť sa k predchádzajúcej verzii

  • Záloha - kód v cloude, nie len na vašom disku
  • Aj marketingové projekty si zaslúžia verziovanie
Git nie je len pre programátorov. Ak pracujete s CC a generujete súbory, chcete mať históriu. Analógia: Ctrl+Z na steroidy.

Git: 3 príkazy

git init

zapni verziovanie v tomto priečinku

git add .

označ všetky zmeny na uloženie

git commit -m "prvý commit"

ulož aktuálny stav s popisom

Minimálne 3 príkazy ktoré potrebujete. git init = zapni Git. git add = vyber čo chceš uložiť. git commit = ulož s popisom. Analógia: init = kúpiť fotoaparát, add = zamieriť, commit = odfotiť.

GitHub

Váš kód v cloude.

  • Záloha — kód v cloude, nikdy nestratíte prácu
  • Nasadenie — web online cez GitHub Pages / Vercel
  • Portfolio — projekty viditeľné pre klientov
Účet na GitHube by mali mať vopred. Ak nemajú, pomôcť vytvoriť. Zdôrazniť praktické benefity: záloha, nasadenie webu, portfolio.

Cvičenie: Git + GitHub

  • git init vo vašom projekte
  • git add . + git commit -m "first commit"
  • Vytvorte repo na GitHub.com
  • git push - pošlite kód do cloudu

Tip: CC vie pomôcť s git príkazmi!

CC môže pomôcť s git príkazmi ak si účastníci nepamätajú syntax. Typické problémy: SSH kľúče, autentifikácia na GitHub. Mať pripravený návod pre HTTPS aj SSH.

Python virtuálne prostredie

Izolácia knižníc pre každý projekt.

Render
